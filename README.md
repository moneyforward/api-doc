# Moneyforward API

クラウド家計簿Moneyforwardの開発者向けAPIのドキュメントです。

## 概要

本APIは現在は、クローズドAPIとして特定のパートナー様へ提供をしております。連携に関心がある方は、下記連絡先よりご連絡ください。

### 主な提供機能

* ユーザ情報の閲覧
* 自動取得金融機関情報の閲覧
* 口座情報の閲覧・編集
* 入出金情報の閲覧・編集
* 資産情報の閲覧

### 認証方式

OAuth2: Authorization Code Flowに基づくOpenID Connect
